# Інформаційна система обліку антикваріату

## Опис

Інформаційна система обліку антикваріату призначена для автоматизації обліку антикварних предметів, категорій та користувачів. Система дозволяє зручно зберігати, редагувати та обробляти дані.

---

## Основні можливості

- реєстрація та авторизація користувачів
- розмежування прав доступу (адміністратор / користувач)
- додавання, редагування та видалення антикварних предметів
- робота з категоріями
- пошук предметів за назвою та категорією
- відображення повідомлень користувачу (flash)
- базове логування дій

---

## Технології

- Python
- Flask
- SQLAlchemy (ORM)
- SQLite
- HTML / CSS

---

## Архітектура проєкту

Проєкт побудований за принципом розділення відповідальностей:

- models/ — класи сутностей (User, Item, Category)
- services/ — бізнес-логіка
- routes/ — обробка HTTP-запитів
- templates/ — HTML-шаблони
- utils/ — допоміжні модулі

---

## Безпека

- паролі зберігаються у хешованому вигляді
- використовується werkzeug.security

---

## База даних

- використовується SQLite
- взаємодія з базою даних реалізована через SQLAlchemy ORM

---

## Запуск проєкту

1. Створити віртуальне середовище:
   python -m venv venv

2. Активувати середовище:
   venv\Scripts\activate

3. Встановити залежності:
   python -m pip install -r requirements.txt

4. Запустити проєкт:
   python run.py

## Особливості реалізації

- використано ORM для роботи з базою даних

- пошук реалізовано на рівні додатку для коректної роботи з кирилицею
